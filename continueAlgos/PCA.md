Principal Component Analysis (PCA)

PCA is a Dimensionality Reduction technique. It is essential when working with datasets that have too many features to process efficiently.

The Concept: It transforms a large set of variables into a smaller one (Principal Components) that still retains most of the original information (variance). It "squashes" the data while preserving the most important patterns.

The Project: High-Dimensional Data Visualization

Goal: Take a complex dataset with 30+ variables and view it on a 2D plot.

Implementation: Apply PCA to a dataset (like genomic data or complex sensor logs) to reduce it to two main components, making it possible to visualize clusters that were previously hidden.

Steps: create the visualization and deploy them on Streamlit
